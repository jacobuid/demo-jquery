# To Run this Project Locally

-   Step 1: download the repository
-   Step 2: Run `npm i`
-   Step 3: Run `php artisan serve`
-   Step 4: (in another terminal) Run `npm run dev`
-   Step 5: Open browser to Localhost:8000 (by default)
